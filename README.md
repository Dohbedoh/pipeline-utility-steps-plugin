# Pipeline Utility Steps Plugin
Small, miscellaneous, **cross platform** utility steps for [Jenkins Pipeline](https://wiki.jenkins-ci.org/display/JENKINS/Workflow+Plugin) jobs.

See the [List of Steps](docs/STEPS.md)
